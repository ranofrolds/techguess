1- O que é o Tech-Guess?


Tech-Guess leva como inspiração o programa televisivo “Mega Senha”, que consiste na adivinhação de palavras para acumular pontuações.


2. Mecânica do Jogo


Envolve dois jogadores, sendo que em cada rodada, um será o adivinhador, enquanto o outro fornecerá dicas para as palavras a serem adivinhadas. As dicas devem consistir em apenas uma palavra por vez, o que significa que, após cada dica, o adivinhador deve responder com uma única palavra antes que uma nova dica possa ser apresentada.


3. Objetivo


O objetivo é acumular pontos ao longo das rodadas. Neste jogo, os pontos são representados de forma metafórica como "memória computacional", começando em 0 bytes e chegando até 1 terabyte.


4. Níveis de dificuldade


Os níveis foram separados com base no possível conhecimento do usuário, sendo o nível 1 para leigos na área, nível 2 aos interessados em tecnologia e o nível 3 possui uma dificuldade referente aos estudiosos do âmbito tecnológico.


5. Balanceamento do jogo



  • Primeira Rodada = { Pontuação: 500 MB; Nível: 1; Nº de palavras a serem adivinhadas: 3; Tempo: 45 s}

  • Segunda Rodada = { Pontuação: 1 GB; Nível: 1; Nº de palavras a serem adivinhadas: 5; Tempo: 1 min}

  • Terceira Rodada = { Pontuação: 10 GB; Nível: 2; Nº de palavras a serem adivinhadas: 3; Tempo: 45 s}

  • Quarta Rodada = { Pontuação: 100 GB; Nível: 2; Nº de palavras a serem adivinhadas: 5; Tempo: 1 min}

  • Quinta Rodada = { Pontuação: 500 GB; Nível: 3; Nº de palavras a serem adivinhadas: 3; Tempo: 45 s}

  • Sexta Rodada = { Pontuação: 1 TB; Nível: 3; Nº de palavras a serem adivinhadas: 5; Tempo: 1 min}
